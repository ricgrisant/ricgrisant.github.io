Inventory App Details

About

This was an App development for the faculty of engineering in the National Autonomous University of Honduras.


Whit this app they can now have better control of the inventory and make more efficient their work and finances.

Features

Features we developed:


Users Control

  • Module to ensure the admin user can control who has access to the app.
  • Admin users can control the permission to erase, add, delete or modify in each module. Whit this they can ensure each user has access only to the actions in the app that the admin considers necessary in order to make their job and nothing more.
  • Each user's actions are saved in the DB to ensure the integrity and security of data. This will make the app easier for a data auditory.

Inventory Control

  • Module to control inventory of the faculty.
  • With 15+ modules, some include:
    • Kardex
    • Warehouse Control
    • Product Entries and Delivery Control
    • Product Entries and Delivery History
    • Product Entries and Delivery PDF Constancy Generator
  • Each user's actions are saved in the DB to ensure the integrity and security of data. This will make the app easier for a data auditory.
